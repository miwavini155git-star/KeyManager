# Инструкция по использованию кастомных скинов

## Как установить скин для игрока:

1. Создайте папку `hunt_skins` в корневой директории игры (рядом с `.minecraft`)
2. Положите PNG файл скина, названный по нику игрока
   - Например: `Dev.png` для игрока с ником "Dev"
   - Например: `Alex.png` для игрока с ником "Alex"

## Требования к файлам скинов:

- Размер: 64x64 пиксела
- Формат: PNG
- Название файла должно соответствовать точному нику игрока (с учетом регистра)

## Пример структуры:

```
hunt_skins/
├── Dev.png
├── Admin.png
├── Player123.png
```

Скины автоматически загружаются при запуске клиента и будут видны для всех игроков на сервере.
